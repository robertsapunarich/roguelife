# roguelife

The codebase for a new rogue-like game, made by three buds.

## demo

[Firebase site](https://roguelife-7f367.firebaseapp.com/).

## dev setup

To install:

```bash
git clone https://github.com/robertsapunarich/roguelife && cd roguelife
npm i
```

To serve for development:

```bash
npm run serve
```

To build:

```bash
npm run build
```
